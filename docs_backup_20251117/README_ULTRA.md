
#### Linux/Mac:
```bash

```

### Method 2: Manual Start

#### 1. Install Dependencies:
```bash
# Server
cd server
npm install

# Client
cd client
npm install
```

#### 2. Setup Environment:
Create `server/.env`:
```env
PORT=5000
MONGODB_URI=mongodb://localhost:27017/laptop-marketplace
JWT_SECRET=your_secret_key_here
NODE_ENV=development
```

#### 3. Start MongoDB:
```bash
mongod
```

#### 4. Start Backend:
```bash
cd server
npm start
```

#### 5. Start Frontend:
```bash
cd client
npm start
```

#### 6. Open Browser:
```
http://localhost:3000
```
